#  Footstep Power Generation

Footstep Power Generation is an innovative renewable energy project that converts mechanical energy produced by human footsteps into electrical energy using piezoelectric sensors. The generated power is rectified, stored, and utilized to operate low-power electrical devices, making it a sustainable solution for energy harvesting in high-footfall areas.

# Project Overview

This project demonstrates how human walking energy can be converted into usable electrical energy. When a person steps on the platform, piezoelectric sensors generate a voltage due to applied pressure. The generated AC voltage is converted into DC using a bridge rectifier, stored in a rechargeable battery, and can be used to power LEDs or other low-power devices.

## ⚙️ Working Principle

1. A person steps on the platform.
2. Piezoelectric sensors generate electrical energy.
3. The generated AC voltage is converted into DC using a bridge rectifier.
4. The DC power is filtered and stored in a rechargeable battery.
5. The stored energy is used to power LEDs and other low-power devices.
